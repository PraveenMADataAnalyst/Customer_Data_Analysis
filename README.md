##Data Analytics Project: Customer & Sales Insights

##Overview

This project shows a complete data analytics workflow: data analysis in Python, SQL queries, Power BI dashboard, and a presentation in Gamma. The goal is to analyze customer and sales data and provide actionable insights.

1.Dataset

Customer, sales, and product data

Columns: Customer ID, Product, Category, Quantity, Price, Order Date, Delivery Time, Feedback

2.Tools

Python (Pandas, NumPy, Matplotlib, Seaborn)

SQL (PostgreSQL / MySQL / SQL Server)

Power BI

Gamma (for presentation)

3.Steps

Load Data – Import dataset in Python

EDA & Cleaning – Handle missing values, remove duplicates, explore trends

SQL Analysis – Run queries for insights (top products, customer segments, sales metrics)

Power BI Dashboard – Create interactive visuals and charts

Reporting – Summarize findings and create PPT in Gamma

4.Dashboard & Results

Visualized sales trends and top products

Analyzed customer segments and repeat buyers

Evaluated delivery performance

Presented insights in an easy-to-understand dashboard and report
